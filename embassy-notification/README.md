# Embassy Notification Script

This script sends diplomatic notifications to embassies of Hague Convention member states.

## Usage

```bash
./send_notifications.sh
```

## Prerequisites

- Python 3.7+
- requests library
- Email/SMTP access

## Files

- `embassies.json` - List of embassies with contact info
- `notifications/` - Draft notifications in multiple languages
